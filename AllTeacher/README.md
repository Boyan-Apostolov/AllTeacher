# AllTeacher

Subscription-based personalized learning app. User states a learning goal in any language, gets a fully adaptive curriculum generated by a pipeline of AI agents.

## Repo

```
backend/       Flask + orchestrator + subagents  (Python 3.12, Docker, Traefik)
ios/           Expo + Expo Router                 (React Native, TypeScript)
SETUP.md       one-time setup walkthrough
```

## Stack

- **Mobile:** React Native + Expo + Expo Router
- **Backend:** Flask in Docker behind Traefik on a home VPS
- **Auth + DB:** Supabase (Postgres + Auth)
- **AI:** OpenAI API (gpt-4o by default)
- **Subscriptions:** RevenueCat + StoreKit

## Get started

Follow [`SETUP.md`](./SETUP.md). It walks through: Python + Xcode, Supabase project + schema, OpenAI API key, booting the backend, running the app in the simulator.

First milestone is a /health round-trip from the iOS simulator to Flask. Everything else (auth, orchestrator, agents, streaming, subscription tiers) builds on top.
